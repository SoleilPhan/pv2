Cooking For One

The goal of this project was to design a magazine spread with any topic of my choice. Because I like to cook in my free time, I chose to write on the topic of college cooking.

Process

I chose an unconventional layout that you would not typically find recipes to be laid out in and I chose to do so because I wanted to create a refreshing approach for the readers. Because the layout was unconventional, choosing the number of columns I wanted to include, the type, the color of the type, and the content I wanted to use was a challenge I faced as a whole. I resolved this problem by thinking about the readability aspect as if I was the reader wanting to be engaged by a magazine article.
